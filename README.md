# My dotfiles

my macos dotfiles in order to setup a new macos machine.

## alacritty
The macos terminal emulator with a modern design. It is very fast and customizable.
The config file is located in `~/.config/alacritty/alacritty.toml` and it is a symlink to this repo.
The theme used is catppuccin-frappe.

## bat
The cat clone with wings. It is a modern replacement for cat with syntax highlighting and git integration.
The config file is located in `~/.config/bat/config` and it is a symlink to this repo.
The theme used is Catppuccin Macchiato.

## bottom
The cross-platform graphical process/system monitor with a customizable interface and a multitude of features.
The config file is located in `~/.config/bottom/bottom.toml` and it is a symlink to this repo.
The theme used is catppuccin-frappe.

## joshuto
The file manager with a modern design and a lot of features.
The config file is located in `~/.config/joshuto/joshuto.toml` and it is a symlink to this repo.

## neofetch
The system information tool with a modern design and a lot of features.

## nvim
The modern text editor with a lot of features and plugins.
The folder contains "AstroNvim", "LazyVim", "NvChad", "kickstart" and "nvim" configurations so that I can switch between them.

## sketchybar
The status bar with a modern design and a lot of features.

## shkd
The hotkey to manage the windows in a tiling way which is inspired by the i3 window manager.

## yabai
The tiling window manager for macos and it is inspired by i3 window manager.

## tmux
The terminal multiplexer with a lot of features and plugins.

## zsh
The modern shell with a lot of features and plugins.
The manager used is oh-my-zsh and the theme used is starship.

The install.sh script will install all the dependencies and the dotfiles.
