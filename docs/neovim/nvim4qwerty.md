# The neovim config for qwerty users

## mode

There are four basic modes for Vim
* Normal-mode
* Insert-mode
* Command-mode
* Visual-mode

### Normal-mode

Normal mode is used to browse files, yank, paste, and delete operations.

In this mode, we are able to press the keyboard to move the cursor quickly. The cursor range from small to large is character, word, line, sentence, paragraph and screen.

After starting vim or neovim, it is in normal mode by default. Regardless of the mode, press the ```<Esc>``` key (sometimes we may need to press it twice, press Esc in the insert mode, it will switch to the normal mode, and after the operation is completed in the command mode or the visual mode, it will automatically enter the normal mode, if you enter the command mode or the visual mode does not perform any operations, press Esc twice) will enter the normal mode.
