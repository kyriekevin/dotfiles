# Neovim

Ambitious Vim-fork focused on extensibility and agility.

## Installation

```brew install neovim```

* This nvim configuration requires Neovim 0.8.0+

After installation, you need to install other apps.

* Install ```python3```, and do ```pip3 install neovim```
* Install nerd-fonts (actually it's optional but it looks nice)

Then, you need to run ```:checkhealth``` command to make sure that all you need is installed.
