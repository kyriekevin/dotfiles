# Homebrew

homebrew is a package manager for macOS.

## Installation

It is difficult to use the ```/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"``` command to install Homebrew without a great VPN.

There is a great way to use this ```/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"``` command to install Homebrew for the Chinese people.

## Usage

After installing Homebrew, you are able to install apps like ```brew install "the name of the app" ```
